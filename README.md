# 🕵️‍♂️ Dev Detective – GitHub User Search & Battle App

Welcome to **Dev Detective**, a modern, responsive web application built with Vanilla JavaScript, HTML, and CSS that integrates seamlessly with the GitHub API. 

Not only can you search and explore developer profiles with a beautiful dark-mode UI, but you can also challenge other developers in **Battle Mode** to see who is the ultimate GitHub champion! 🏆

---

## ✨ Features

### 🔍 User Search Mode
* **Real-time API Integration:** Instantly fetch and display data from `api.github.com`.
* **Beautiful Profile Cards:** Display key developer details including Avatar, Name, Bio, Followers, Following, and Repositories.
* **Latest Repositories:** Automatically fetches and displays the user's top 5 most recently updated repositories along with their tech stack and star counts.
* **Resilient States:** Smooth loading spinners and friendly error UI when a dev isn't found.

### ⚔️ Battle Mode
* **Head-to-Head Comparison:** Enter two GitHub usernames and click "Commence Battle".
* **Followers Showdown:** The app evaluates both profiles and mathematically crowns the winner based on Total Followers.
* **Dynamic Progress Track:** Custom animated visual bars mathematically scale to show the precise percentage breakdown of follower dominance.
* **Visual Triumphs:** The winner's card receives glowing victory aesthetics and a trophy, while the runner-up is respectfully dimmed to grayscale. 

### 💅 Premium UI / UX
* **100% Responsive Framework:** Flawlessly adapts to desktop, tablet, and mobile viewing.
* **SaaS Grade Design:** Incorporates glowing hover states, deep box-shadow nesting, modern Google Inter typography, and sweeping entrance animations (`slideInLeft` / `slideInRight`).

## 🚀 Getting Started

Since it's built with Vanilla web technologies, getting started is completely effortless!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ayush-ranjan9135/GitHub-User-Search.git
   ```
2. **Open the project:**
   Simply open the `index.html` file in your favorite modern web browser (Chrome, Firefox, Safari, Edge).
3. **No Build Step Required!** Zero `npm install` or dependency management needed.
